# 魔力兑换助手 (数据探索版)

这是一个用于探索Moviepilot系统中可用数据的测试插件，专注于分享率监控功能。

## 功能特性

- ✅ **站点选择**: 支持选择多个站点进行监控
- ✅ **分享率监控**: 监控站点分享率，低于阈值时触发操作
- ✅ **数据探索**: 详细打印获取到的站点数据信息
- ✅ **定时执行**: 支持cron表达式定时执行
- ✅ **通知系统**: 支持执行结果通知

## 当前版本限制

⚠️ **注意**: 当前版本为数据探索版本，仅用于：
- 获取和显示站点数据
- 验证数据获取流程
- 测试分享率监控逻辑
- **不执行实际的魔力兑换操作**

## 安装和使用

1. 将插件文件夹复制到Moviepilot的插件目录
2. 在Moviepilot后台启用插件
3. 配置以下参数：
   - **启用插件**: 开启插件功能
   - **站点列表**: 选择要监控的站点
   - **执行周期**: 设置监控频率
   - **启用分享率检查**: 开启分享率监控
   - **分享率阈值**: 设置触发操作的阈值

## 数据探索

插件会详细打印获取到的站点数据，包括：
- 站点名称和数据日期
- 分享率数值
- 上传/下载量
- 其他可用字段

通过查看日志可以了解Moviepilot系统中可用的数据字段和格式。

## 后续开发

## 技术架构

- 基于Moviepilot插件标准架构
- 使用dataclass进行配置管理
- 支持事件驱动和定时任务
- 模块化设计，便于扩展
